**********说明********************
1. 此分支下为发布的所有版本的软件，包含更新说明和记录。

2. rundir文件夹下为最新版本的运行环境。如无特别说明，将执行文件放到rundir文件夹下，即可运行。运行环境不同的旧版本，无需移动，更新文件夹内包含运行所需文件。

**********关于软件版本************
采用的软件版本格式为：
V主版本号.子版本号.修正版本号.编译版本号.阶段+发布版本号.日期版本号
其中阶段为内部测试阶段（α），外部测试阶段（β），正式发布阶段（γ）。
各部分含义说明详见word文档，软件版本.docx。下面有简单举例说明


----内部测试：V主版本号.子版本号.修正版本号.编译版本号.α发布版本号.日期版本号
Eg:V3.0.0.14393b64. a0.190410
表示主版本号3，子版本号0，修正版本号0，编译版本号14393b64，内部测试版，发布版本号0，发布日期为19年4月9日
其中编译版本号表明，目标编译平台为14393，64位。


----外部测试：V主版本号.子版本号.修正版本号.编译版本号.β发布版本号.日期版本号
Eg：V3.0.0.14393b64. b0.190410
表示主版本号3，子版本号0，修正版本号0，编译版本号14393b64，外部测试版，发布版本号0，发布日期为19年4月9日
其中编译版本号表明，目标编译平台为14393，64位。


----发布版本：V主版本号.子版本号.修正版本号.编译版本号.γ发布版本号.日期版本号
Eg：V3.0.0.14393b64. r0.190410
表示主版本号3，子版本号0，修正版本号0，编译版本号14393b64，正式发布版，发布版本号0，发布日期为19年4月9日
其中编译版本号表明，目标编译平台为14393，64位。
